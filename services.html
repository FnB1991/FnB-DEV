HTML 5


HTML, ou HyperText Markup Language, é a linguagem fundamental para a criação de páginas web. Composta por tags que estruturam o conteúdo de forma hierárquica, o HTML define a aparência e organização das informações online, sendo a base essencial para o desenvolvimento de sites e aplicações na internet.


JavaScript


JavaScript é uma linguagem de programação versátil e dinâmica, amplamente utilizada para criar interatividade em páginas web. Com recursos como variáveis, operações em strings e eventos, o JavaScript permite desde armazenar valores até responder a ações do usuário, tornando as páginas mais dinâmicas e funcionais.


CSS 3


CSS 3 é a evolução da linguagem CSS, trazendo recursos avançados para estilizar páginas web de forma dinâmica. Com capacidade para criar animações em 2D e 3D, oferece controle detalhado sobre o estilo dos elementos das páginas, permitindo layouts modernos e interativos, elevando a experiência do usuário na web.
